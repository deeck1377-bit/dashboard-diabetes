# Conoce tu Diabetes · Dashboard educativo

Dashboard interactivo para pacientes con diabetes — **Sesión 1: Generalidades de la diabetes**.

Programa de educación en diabetes dirigido por la **Dra. Luisa Fernanda Díaz Domínguez**.
Hecho en un modelo activo por **PromptKilla Studio**.

## Actividades
1. 🔑 **La llave y la cerradura** — simulador de insulina, glucosa y células
2. 🃏 **Tarjetas comparativas** — diabetes tipo 1, tipo 2 y gestacional
3. 🧭 **Velocímetro glucémico** — metas de control, caso clínico y mini evaluación

## Stack
Vite + React. Para correr localmente:

```bash
npm install
npm run dev
```

> Material educativo. No sustituye la consulta médica; las metas mostradas son generales (ADA) y las metas personales las define el equipo de salud de cada paciente.
