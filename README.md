# Conoce tu Diabetes · Dashboard educativo

Dashboard interactivo para pacientes con diabetes, con portada general y dos módulos:

- **Trabajo 1 · Generalidades de la diabetes** (Sesión 1 de 3)
- **Trabajo 2 · Resolver problemas** (Sesión 3 de 3)

Programa de educación en diabetes.
Hecho en un modelo activo por **PromptKilla Studio**.

## Trabajo 1 · Generalidades
1. 🔑 **La llave y la cerradura** — simulador de insulina, glucosa y células
2. 🃏 **Tarjetas comparativas** — diabetes tipo 1, tipo 2 y gestacional
3. 🧭 **Velocímetro glucémico** — metas de control, caso clínico y mini evaluación

## Trabajo 2 · Resolver problemas
1. 🍬 **Hipoglucemia** — Regla del 15-15 interactiva y tarjetas ¿hipo o hiper?
2. 🚦 **Hiperglucemia** — semáforo de decisiones con escenarios
3. 🤒 **Días de enfermedad** — infografía interactiva y juego ¿Qué harías si...?
4. 📝 **Evaluación sumativa** — cuestionario virtual con puntaje

## Stack
Vite + React. Para correr localmente:

```bash
npm install
npm run dev
```

> Material educativo. No sustituye la consulta médica; las metas mostradas son generales (ADA) y las metas personales las define el equipo de salud de cada paciente.
